# Admin Guide - Horizon Bank Admin Dashboard

## Accessing Admin Dashboard

1. **Login** with admin credentials
2. **Enable Admin Mode** in sidebar (checkbox)
3. **Click "Admin Dashboard"** or "Audit Reports"

## Admin Dashboard Features

### System Health Overview
- Total Applications
- Pending Review
- Pending Documents
- Last 24h Applications
- Application Status Breakdown

### Pending Applications Tab
- View all pending KYC applications
- See customer details
- Check document counts
- Filter and sort applications

### Fraud Alerts Tab
- View suspicious activity
- Applications with rejected documents
- Multiple application attempts
- Review flagged cases

### Verify Applications Tab
1. **Enter Application ID** to verify
2. **View Application Details:**
   - Customer information
   - Application status
   - Uploaded documents

3. **Verify Documents:**
   - Click "Verify" on each document
   - Add verification notes
   - Update document status

4. **Update Application Status:**
   - Select new status
   - Add notes
   - Click "Update Status"

**Available Statuses:**
- Under Review
- Document Verification
- Approved
- Rejected
- Pending Resubmission

### Analytics Tab
- Application trends (line chart)
- Date range filtering
- Status breakdowns
- Performance metrics

## Audit Reports Features

### Audit Logs Tab
- View all system activities
- Filter by date range
- Filter by action type
- Export to CSV/Excel

### Compliance Report Tab
- Generate comprehensive reports
- Date range selection
- Application statistics
- Document statistics
- User statistics
- Audit log summaries

### Application Trail Tab
- Enter Application ID
- View complete audit trail
- Export trail to CSV
- Track all actions on application

## Best Practices

1. **Review Applications Regularly**
   - Check pending applications daily
   - Verify documents promptly
   - Update statuses in timely manner

2. **Monitor Fraud Alerts**
   - Review flagged cases immediately
   - Investigate suspicious patterns
   - Take appropriate action

3. **Generate Reports**
   - Weekly compliance reports
   - Monthly audit summaries
   - Quarterly analytics review

4. **Document Verification**
   - Verify all mandatory documents
   - Add clear verification notes
   - Update status accurately

## Security

- Admin access is role-based
- All actions are logged in audit trail
- Export permissions required
- Session management enforced
