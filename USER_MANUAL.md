# User Manual - Horizon Bank KYC Portal

## Getting Started

### Registration Process

1. **Click "Open Account"** on the homepage
2. **Fill Personal Information:**
   - First Name* and Last Name*
   - Email Address* and Phone Number*
   - Date of Birth* and Gender*
   - Marital Status* and Age*

3. **Fill Employment Details:**
   - Occupation*
   - Monthly Salary*
   - Annual Income*

4. **Create Account Credentials:**
   - Choose Username*
   - Create Password* (minimum 8 characters)
   - Confirm Password*

5. **Enter Address:**
   - Residential Address*
   - City/Town*
   - Pincode*

6. **Agree to Terms & Conditions***
7. **Click "Create Account"**

**Note:** After registration, your account is created with KYC Status: "Not Submitted". You'll need to complete KYC verification on your first login.

---

### First Login & KYC Verification

1. **Login** with your username/email and password
2. **You'll be redirected to KYC Portal** (if KYC not submitted)

3. **Upload Identity Document (Mandatory):**
   - Select document type (Aadhar, PAN, Passport, or Voter ID)
   - Upload a clear scan (PDF, JPG, PNG)
   - OCR will automatically verify the document

4. **Upload Photo (Mandatory):**
   - **Option 1:** Upload Photo - Browse and select a photo file
   - **Option 2:** Use Webcam - Click to activate camera and take a live photo
   - If webcam doesn't work, use "Upload Photo" option

5. **Additional Information (Optional):**
   - PAN Card Number
   - Aadhar Number
   - Nominee Name
   - Nominee Relation
   - OTP Verification checkbox

6. **Click "Submit KYC Application"**

**After Submission:**
- Your KYC Status changes to "Submitted"
- Application is sent for review
- You'll receive notifications on status updates

---

## Troubleshooting

### Can't Login?
- Check if account exists (use "Check Status")
- Verify username/email is correct
- Ensure password is correct
- Contact support if issues persist

### Webcam Not Working?
- Check browser permissions for camera
- Use "Upload Photo" option instead
- Ensure you're on HTTPS or localhost

### Document Upload Failed?
- Check file size (max 10MB for documents, 2MB for photos)
- Ensure file format is supported (PDF, JPG, PNG)
- Try uploading again

### KYC Status Not Updating?
- Wait a few minutes and refresh
- Check your dashboard
- Contact support if issue persists
